Web Development Concepts:

Definitions of terms:

Website: A program or html document hosted on a server that can be delivered over the internet. It is static.

Web Application: A program hosted on a remote server that can be delivered over the internet. It is dynamic.

Web Browser: A program that makes HTTP requests to URLs to interpret and display the HTML it recieves.

Web Server: Server programs written in various languages.

HTTP Request: A protocol that describes how a client can retrieve html documents from a server using the document's URL.

HTTP Response: A protocol that describes how a server can send html documents to a client that requested the document's URL.

Frontend Design:

The concept of frontend design

Effective
A website should help users meet thier goals.
Efficient
A website should not make users take extra steps to thier goal.
Easy to navigate
A website should be intuitive and allow users to understand how to use its functions.
Error-free
A website should avoid common user errors and be designed to anticipate potential roadblocks.
Engaging
A website should fit the user's needs and fit the previous criteria. Will they come back to use this website more?

The purpose of Layout Tags:

Layout Tag Function
Header Defines a header for a document or a section
Nav Defines a set of navigation links
Section Defines a section in a document that holds articles
Article Defines independent, contained content
Aside Defines content at the side, where a side bar would be
Footer Defines additional details that is openable and closable
Summary Defines a heading for the details element
