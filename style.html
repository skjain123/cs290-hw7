Styles

Locations

There are three different locations that you can apply CSS styles: .

  • Inline : These are applied directly to the HTML element using the style attribute p style="color: red;"> /p
  • Internal : These are applied inside the style tags, style p { color: redpx; } /style
  • External : These are applied in a separate document linked by-- link rel="stylesheet" href="styles.css">

Selectors

There are 4 different selectors: Class, Id, Attribute, pseudo-selectors

  • Class-- can be used multiple times in a document, used in CSS with a "." before it -- .button {background-color: blue;}
  • Id -- can only be used once per document, used in CSS with a "#" before it -- #example {color: red;}
  • Attribute -- target elements using thier attributes -- [type="text"] {border: 1px solid black;}
  • Pseudo-Selector -- target elements using their states -- :hover, :active, :first-child

Box Model

This describes the rendering of the elements on a webpage-- content, padding, border, margin.

  • Content : The actual content of the element, like the text or image.
  • Padding : The space between the content and the element's border.
  • Border : The border that surrounds the padding.
  • Margin : The space outside the border.

Inheritance & Specificity

  • Inheritance : CSS properties can be inherited from parent elements to children elements. Ex: fonts.
  • Specificity : Determines what CSS property applies when there are conflicting properties in inheritance and CSS selectors.
